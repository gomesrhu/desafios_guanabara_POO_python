# Desafios de Programação Orientada a Objetos (Python)
Este repositório contém as soluções dos desafios propostos na playlist de Programação Orientada a Objetos com Python do canal Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

O objetivo deste projeto é exercitar os pilares da POO (Abstração, Encapsulamento, Herança e Polimorfismo) utilizando a linguagem Python.
